

# Metaseq
A codebase for working with [Open Pre-trained Transformers](projects/OPT), originally forked from [fairseq](https://github.com/facebookresearch/fairseq).


## Install
"""
pip install -e .
"""
Optional:
```
pip install fairscale
```
